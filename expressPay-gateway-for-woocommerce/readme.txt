This plugin allows you to accept payments through ExpressPay on your WooCommerce store.

== Description ==
Easy to integrate to accept secure payments using Mobile Money and Credit/Debit Cards.

== Features ==
Securely accept payments through ExpressPay.
Easy configuration within the WooCommerce settings.

== Installation ==
*   Go to __WordPress Admin__ > __Plugins__ > __Add New__ from the left-hand menu
*   In the search box type __ExpressPay Gateway for WooCommerce__
*   Click on Install now when you see __ExpressPay Gateway for WooCommerce__ to install the plugin
*   After installation, __activate__ the plugin.
*   Go to WooCommerce Settings -> Payment Gateways and configure your ExpressPay Gateway for WooCommerce settings.
